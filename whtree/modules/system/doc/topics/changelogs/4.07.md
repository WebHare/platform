# 4.07
## Consilio
- Initial Elasticsearch support

## Tollium
- PostInitComponent has been deprecated, try to use PreInitComponent or PreShowComponent. PreShowComponent will run just before a component is sent to the client, and
  is more useful than PostInitComponent when screens are opened from Init().

## Website and module development
- The Publisher forms api now supports RTDs and image edit fields
- Experimental (development servers only) publisher global 'search' feature
- Versioning policies to control who is allowed to change/publish website content

## WebHare development
- OCI finally supports 64bit macOS. See [Oracle support/OCI driver](databases/oracle.md) for more information on how to enable.
