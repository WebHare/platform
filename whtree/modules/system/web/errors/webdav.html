WebHare WebDav support

The URL you requested lies inside the WebDav namespace of an installed WebHare webserver, and requires that you access it through the WebDav protocol.

  • If you use Windows Explorer, you should select "Add Network Place", which you can find using the Network Neighbourhood or My Network Places icon on your desktop
  • In Internet Explorer, you should use the "Open Web Folder" option when opening an URL. You can select "File, Open" or use CTRL+O in Internet Explorer to access the "Open URL" dialog
If you have any problems accessing the WebDav interface, please contact your system administrator.
